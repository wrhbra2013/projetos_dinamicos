# Tutorial - Configuração Stripe Checkout

## 1. Criar Produtos

1. Acesse o Dashboard da Stripe: https://dashboard.stripe.com
2. No menu lateral, clique em "Products"
3. Clique em "Add product" para cada plano:

### Plano Pro
- Name: Projetos Dinâmicos Pro
- Description: Plano Pro com recursos ilimitados
- Pricing: R$ 29,00 por month
- Currency: BRL

### Plano Empresarial
- Name: Projetos Dinâmicos Empresarial
- Description: Plano empresarial com múltiplos usuários
- Pricing: R$ 99,00 por month
- Currency: BRL

## 2. Obter Links de Checkout

### Opção A: Checkout Hosted (Sem backend)

1. Na lista de produtos, clique no produto criado
2. Clique em "Create checkout session"
3. Configure:
   - Pricing: Monthly
   - Payment method: Card
   - Allow promotion codes: Sim
4. Clique em "Create"
5. Copie o link gerado (algo como: https://buy.stripe.com/...)

### Opção B: Usar API (Com backend)

```javascript
const stripe = require('stripe')('sk_test_XXXXX');

const session = await stripe.checkout.sessions.create({
  payment_method_types: ['card'],
  line_items: [{
    price: 'price_XXXXXXXXXXXXXX', // ID do produto (ex: price_123...)
    quantity: 1,
  }],
  mode: 'subscription',
  success_url: 'https://seudominio.com/sucesso',
  cancel_url: 'https://seudominio.com/planos',
});

console.log(session.url); // Link para redirecionar o usuário
```
