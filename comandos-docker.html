📦 Comandos Docker

Amor Animal — gerenciamento de containers no servidor Debian

ℹ️ Containers
nome amoranimal-api
nome amoranimal-db
📋 Status
Listar containers em execução
docker ps
Status detalhado dos serviços
docker compose ps
📝 Logs
API — últimas 50 linhas
api docker compose logs --tail=50 api
API — tempo real
api docker compose logs -f api
Banco — últimas 50 linhas
db docker compose logs --tail=50 db
Banco — tempo real
db docker compose logs -f db
Todos os serviços — tempo real
all docker compose logs -f
Logs pelo nome do container (forma direta)
api docker logs amoranimal-api --tail 50
db docker logs amoranimal-db --tail 50
api docker logs -f amoranimal-api
⚙️ Gerenciamento
Reiniciar a API
docker compose restart api
Reconstruir e recriar containers
docker compose up -d --build
Parar todos os containers
docker compose down
Parar e remover volumes (dados do BD serão perdidos)
docker compose down -v
💻 Shell no container
Acessar o terminal da API
docker exec -it amoranimal-api sh
Acessar o terminal do banco
docker exec -it amoranimal-db sh
Acessar o PSQL diretamente
docker exec -it amoranimal-db psql -U postgres -d amoranimal_db
💡 Dicas
  • Execute os comandos no diretório onde está o docker-compose.yml
  • Se precisar de sudo, anteponha ao comando
  • Ctrl+C sai do modo follow (-f)
