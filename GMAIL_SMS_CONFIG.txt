===========================================
CONFIGURAR GMAIL PARA SMS - Projetos Dinâmicos
===========================================

1. ATIVAR VERIFICAÇÃO EM 2 ETAPAS:
   - Acesse: https://myaccount.google.com/signinoptions/two-step-verification
   - Ative a verificação em duas etapas

2. CRIAR SENHA DE APP:
   - Acesse: https://myaccount.google.com/apppasswords
   - Selecione "Outro" e nomeie (ex: "ProjetosDinamicos")
   - Copie a senha de 16 caracteres gerada

3. CONFIGURAR NO PROJETO:
   - Edite o arquivo .env (ou crie um)
   
   DATABASE_URL=postgresql://postgres:wander@localhost:5432/projedydb
   SECRET_KEY=sua_chave_secreta_aqui
   GMAIL_EMAIL=seuemail@gmail.com
   GMAIL_PASSWORD=xxxx xxxx xxxx xxxx

4. OPERADORAS SUPORTADAS NO CADASTRO:
   - Vivo    -> numero@vivo.com.br
   - Claro   -> numero@claro.com.br
   - TIM     -> numero@tim.com.br
   - Oi      -> numero@oi.com.br
   - Nextel  -> numero@nextel.com.br

5. RODAR O PROJETO:
   pip install -r requirements.txt
   python app.py

6. TESTAR:
   - Acesse http://localhost:5001
   - Faça o cadastro
   - Veja no console: [GMAIL SMS] Enviado para...

NOTAS:
- O Gmail SMTP é gratuito
- A operadora pode cobrar pelo recebimento do SMS
- Sem configuração, funciona em modo simulação (console)
