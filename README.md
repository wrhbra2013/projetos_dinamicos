# CodeTrain - Plataforma de Treinamento em Programação

Uma plataforma interativa para aprender e praticar programação em várias linguagens.

## Linguagens Disponíveis

- **Python** - 🐍
- **JavaScript** - 🌐
- **Java** - ☕
- **C++** - ⚡
- **Go** - 🔷

## Como Usar

Abra o arquivo `index.html` diretamente no navegador, ou use um servidor local:

```bash
# Python (recomendado)
python3 -m http.server 8000

# Node.js
npx serve .
```

Acesse: http://localhost:8000

## Estrutura do Projeto

```
codetrain/
├── index.html              # Página principal
├── static/
│   ├── css/
│   │   └── training.css    # Estilos da plataforma
│   └── js/
│       └── training.js     # Lógica e exercícios
└── README.md
```

## Funcionalidades

- **Editor de código** com syntax highlighting
- **Execução de JavaScript** diretamente no navegador
- **Sistema de progresso** com localStorage
- **Conquistas** para motivar o aprendizado
- **Modo tela cheia** paraتمرين focado
- **Soluções** disponíveis após completar exercícios

## Exercícios

Cada linguagem possui exercícios organizados por dificuldade:
- **Fácil** - Conceitos básicos
- **Médio** - Algoritmos e estruturas de dados
- **Difícil** - Problemas avançados

## Tecnologias

- HTML5, CSS3, JavaScript (Vanilla)
- localStorage para persistência de dados
- Sem dependências externas (exceto Google Fonts)
