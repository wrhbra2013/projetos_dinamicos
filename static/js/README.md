Baixe o arquivo `mermaid.min.js` na versão desejada e coloque aqui como `mermaid.min.js`.
Exemplo (na sua máquina):

curl -L https://cdn.jsdelivr.net/npm/mermaid/dist/mermaid.min.js -o static/js/mermaid.min.js

Se preferir usar o CDN, não precisa colocar o arquivo local; o loader faz fallback automaticamente.
